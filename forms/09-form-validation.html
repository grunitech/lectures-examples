Pattern based validation (numbers only)

  • \d+
  • \d represent any digit
  • + represent one or more character

Pattern based validation (english only)

  • [a-zA-Z]+
  • [] group
  • a-z all small letters from a to z
  • A-Z all capital letters from a to z

Pattern based validation (hebrew only)

  • [א-ת]+
  • Another way is to use UNICODE notation
  • Beit letter: ב
  • Beit letter: ב
  • We can write Aleph to Tav using: [\u05D0-\u05EA]+
